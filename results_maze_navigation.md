## Setup as actually run

- **Data**: 8×8 grids generated by recursive backtracking; ~100,000
  mazes, ~1.5M training tokens after BOS/EOS framing. Three
  conditions: `real` (shortest paths), `within_shuffled` (token order
  shuffled within each path; cell-identity set preserved),
  `global_shuffled` (cell tokens reassigned uniformly across the
  corpus; cell-identity destroyed).
- **Model**: nanoGPT-style, n_layer=6, n_head=6, n_embd=192,
  block_size=64, ~2M parameters. As locked.
- **Training**: standard nanoGPT loop; train/val/gen split 80/10/10;
  gen is maze-level (test mazes the model never saw walls of).
- **Evaluation splits**:
  - **position-level** (random 80/20 over probe positions)
  - **maze-level honest** (test mazes never seen during training);
    this is the load-bearing split.
- **Seeds**: 5 (outermost loop varying untrained init + activation
  sampling positions + probe-training RNG).
- **Probes**: linear + MLP per layer at residual stream;
  cross-entropy loss; class-balanced sampling where applicable.
- **Transplant**: per-layer (L0…L5) residual-stream substitution;
  metric is "P(B's next path step)" minus the random-control baseline,
  5 seeds × 6 layers × 3 conditions = 90 transplant runs.

| P4 | Starting cell NULL, gap ≤ 0.10 | ✗ **falsified** (null direction; gap +0.15) |

**Both pre-registered substantive predictions (P3 and P4) failed in
opposite directions.** Distance was predicted encoded and was not.
Starting cell was predicted null and was encoded. The strict
N-criterion does not survive this domain.

## What this teaches us about the N-criterion

The two substantive falsifications point to a single deeper issue:
the strict N-criterion conflated **"required for next-token
prediction"** (a necessity/sufficiency claim about the task) with
**"recoverable from the residual stream"** (an empirical claim about
the model). These are different things.

**On the null direction (P4 falsification)**: The starting cell sits
in a positionally distinct slot (the first non-BOS token), and
self-attention can copy that slot's content forward across the
sequence essentially for free. The model has no incentive to actively
*discard* the starting cell — predictive irrelevance does not imply
representational erasure. Information that is already present and
attention-accessible persists by default.

**On the positive direction (P3 falsification)**: The model can
predict the next path step using current cell + local prefix without
ever building an explicit "distance to goal" feature in the residual
stream. "Useful for the task" does not entail "explicitly represented
as a feature." The model can compute distance implicitly inside MLP
layers or use cell-coordinate proxies without a clean linear feature
to probe.

**The graded form of the N-criterion (architectural carry-through)
survives the P4 falsification** as a specific mechanistic
addition: any feature whose input tokens are attention-accessible
and whose computation is *cheap given attention* (e.g., a single copy
operation from a positionally distinct slot) will persist in the
residual stream regardless of its predictive necessity. This is the
mechanism named by the pivot from strict to graded forms after the
maze result.

**On non-monotone destroyed-structure (real ≈ global > within)**:
The original framing assumed `real > within > global` monotonicity
on every feature, reading destroyed-structure as a continuum of
"removed structure." The actual finding is that *the encoded
feature is sequential co-occurrence structure*: global-shuffling
preserves which tokens follow which (the surface bigram statistics);
within-shuffling destroys it. The destroyed-structure axis is
two-dimensional (cell-identity × sequential-cooccurrence), not
one-dimensional. The strict gradient prediction was wrong because
the test variants vary along different axes.
